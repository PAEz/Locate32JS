    Some js for reading locate32 db files.

    Project maintained by PAEz Hosted on GitHub Pages — Theme by mattgraham

    Intro

    This js will allow you to read from a database file generated by locate32 for windows. Locate32 is a great app for windows that indexs all your files into a db to allow for faster searching. Ive always wanted access to this db and now I have it. Was written in and only tested on Chrome, very happy with the speeds its getting.
